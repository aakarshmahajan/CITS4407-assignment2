## What is this project

This assignment is about cleaning and analysing YouTube trending
video data using Bash shell scripts.

## Files

- clean : script to clean the data
- analyse : script to analyse the data

## Data

The input file is trending_videos_unclean.csv which has 7 columns:
video_id, publish_date, views, likes, dislikes,
comments_disabled, ratings_disabled

## How to run

./clean trending_videos_unclean.csv
./analyse trending_videos_clean.csv
